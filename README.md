# QuickBite - Frontend

Modern food delivery application built with React and Vite.

## Features
- Role-based dashboards (Customer, Restaurant, Rider)
- Real-time order tracking with maps
- Shopping cart
- Delivery address management
- Beautiful, responsive UI

## Tech Stack
- React + Vite
- React Router
- Leaflet for maps
- Axios for API calls
- Plain CSS with modern design

## Development
```bash
npm install
npm run dev
```

## Deployment
Configured for Vercel. The app automatically connects to the backend API in production mode.
